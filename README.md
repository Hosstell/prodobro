# Pro dobro

Проект был реализован в финале хакатона "Цифровой прорыв 2019" и последующей акселерации командой "R34".

**Стек:**
1) Django
2) Vue.js
3) GraphQL (backend: Graphene, front: Apollo)
4) PostgreSQL
5) redis
6) Docker, Docker-compose

## Запуск

Frontend
```bash
npm install
npm run dev
```

Backend
```bash
pipenv install
pipenv shell
python manage.py runserver
```
